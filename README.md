# Cosmic Survivors
Cosmic Survivors is an arcade-style survival game developed and published by **Super Star Interactive**.  
This repository hosts the official builds, releases, and technical project files for the game.

---

## 🪪 About the Game
Cosmic Survivors is a fast, minimalist arcade challenge where you pilot a small rocket through an field of obstacles.  
Dodge incoming debris, stay focused, and push your high score as far as your reflexes allow.  
One hit ends the run so every second counts.

---

## 📖 Features
- Smooth and responsive 2D rocket controls  
- Randomized obstacle spawning for endless replayability  
- High score system  
- Clean, retro‑inspired visuals  
- Lightweight, pick‑up‑and‑play design  

---

## 🛠️ Systems and Technology
Cosmic Survivors is built using:

- **Unity 6000.3.8f1**
- C# scripting
- Custom-made assets and effects
- Unity UI Toolkit for interface elements

Cosmic Suvivors uses:
- SSVNS 2, the version numbering system made by Super Star Interacive. Read the SSVNS2.md file for more information
